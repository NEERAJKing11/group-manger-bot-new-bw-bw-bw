# Board Wallah Attendance Bot

## Commands
- `/register Name` : Student khud ko register kare.
- `/end_day` : Raat ko admin run karega. Ye check karega ki aaj kisne 1 bhi quiz nahi di aur unhe strike dega.

## How to use
1. Quiz ka result group me forward karein (Attendance lag jayegi).
2. Raat ko sone se pehle `/end_day` daba dein (Absent walo ko strike milegi).
